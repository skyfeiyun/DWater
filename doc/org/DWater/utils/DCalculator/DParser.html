DParser is a Class for formula parsing. You can simply create a DParser Object, pass your formula string to DParser compiler. After compiling this formula you can pass the value of variable to the function calculate() to calculate the result.

DParser()Constructor
public function DParser(originCommand:String, paramName:Array)

Create a new formula parser, for one formula you need one parser related to it. The formula string is not case sensitive.

Parameters
originCommand:String — A formula String you should pass to the constructor, such as x+3, these operator is support in the parser:

+(add), -(substract), multiple), /(divide), ^(pow), abs, acos, asin, atan, sin, cos, tan, floor, ceil, round, max, min, log(base 2).

 
paramName:Array — If your formula string contains a series of variable. Pass the varibale name using array of string.
Method Detail
calculate()method
public function calculate(paramArray:Array):Number

Parameters

paramArray:Array — The variable parameters array. For example, if you have 2 variable x=1,y=2, you should pass the variable using this format:[{name:"x",value:1},{name:"y",value:2}]

Returns
Number — The result of this formula.

Example
The following code show how to calculate the formula "x+y": 
             var parser:DParser=new DParser("x+y",["x","y"]);
             var result:Number;
             result=parser.calculate([{name:"x",value:3},{name:"y",value:4}]);
             trace(3,",3,"+",4,",4,"=",result);
